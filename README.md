# MindLog (Mood Insight and Narrative Diary Log)
## Deskripsi Proyek
MindLog adalah aplikasi mood tracking yang membantu pengguna mengenali dan mengekspresikan emosi, serta memberikan feedback sederhana untuk mendukung pemahaman dan pengelolaan perasaan.
## Fitur Utama
1. Input Mood Harian
Pengguna dapat memasukkan data berupa:
-Skala mood (1–10)
-Alasan dari kondisi emosi
-Catatan singkat (journaling)
2. Penyimpanan Data
Data yang diinput akan disimpan dalam file sederhana (CSV/JSON) sehingga dapat digunakan kembali untuk melihat riwayat.
3. Riwayat Mood (History)
Menampilkan data mood pengguna berdasarkan waktu untuk melihat perubahan emosi dari hari ke hari.
4. Insight Sederhana
Aplikasi akan memberikan analisis ringan seperti:
-Rata-rata mood
-Mood tertinggi dan terendah
5. Feedback Otomatis
Sistem memberikan respon sederhana berdasarkan kondisi mood pengguna, seperti:
-Saran istirahat saat mood rendah
-Motivasi saat mood tinggi
